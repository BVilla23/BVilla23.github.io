Project 1


HTML Project: Dreaming Places

Project 1 was about creating an informative website. You will find a lot of information about Brazil. Information such as what Brazil is known for, the famous Cristo Redentor statue, and Copacabana Beach can be found on the page. In addition to recent news you will also find the cost per day for a trip to Brazil.

I created a website about Brazil because it's a place I would love to visit one day. It's tropical climate and colorful scenery was something that always caught my eye.

Project 2


HTML and CSS Project: Sales Website

Project 2 is a sales webpage. The idea behind the project was to create a sales page that had animation, slideshow, and modal windows using CSS.

The sales page was a imaginary body care company. I created sections to showcase single and gift products. Each product had its description and a modal window. I also maintain the page alive adding a slideshow and an animation to the logo.
